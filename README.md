## Використання:
Додаток дозволяє записувати список справ, які будуть зберігатись на онлайн сховищі. Свої справи можжна переглянути з будь-якого пристрою, якщо увійти у свій акаунт.

## Технології:
- Android
- Kotlin
- Firebase
